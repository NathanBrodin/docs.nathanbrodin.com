# Windows Subsystem for Linux (WSL) Installation
Hey there! In this page, we'll be focusing on the Windows Subsystem for Linux (WSL) and how to install it on your Windows 11 system. By the end, you'll have a powerful tool to help you work with Linux-based environments seamlessly.

First things first, just follow [this link](https://learn.microsoft.com/fr-fr/windows/wsl/install) to get detailed instructions for setting up WSL on your Windows 11 machine.

<Callout>
    While you're at it, don't forget to enhance your WSL experience by adding [Oh my posh](https://ohmyposh.dev/).
</Callout>
Once you have WSL up and running, here are some useful packages you might want to install using `apt install`:

- tree
- gcc
- python
- cbonsai (try it out with cbonsai --live)
